# 🌌 Acerca del proyecto

**Event Rental Frontend** es una plataforma web enfocada en la administración y reservación de eventos sociales como bodas, fiestas, reuniones corporativas y celebraciones privadas.

La interfaz fue diseñada para brindar una experiencia moderna, intuitiva y responsive para clientes y administradores.

El sistema permite:

- 🎉 Gestionar eventos
- 📅 Realizar reservaciones
- 👥 Administrar clientes
- 🪑 Rentar mobiliario y decoración
- 💳 Gestionar pagos
- 📦 Administrar paquetes
- 📊 Supervisar servicios
- 🌐 Facilitar la organización de eventos

# 👨‍💼 Módulos del sistema

## 🎉 Event Module

Este módulo administra los eventos registrados en la plataforma.

### Funcionalidades:

- ➕ Crear eventos
- 📅 Programar fechas
- 📍 Seleccionar ubicaciones
- 📋 Gestionar detalles
- 📊 Supervisar eventos

---

## 🪑 Rental Module

Este módulo administra productos y servicios para renta.

### Funcionalidades:

- 🏕️ Gestión de carpas
- 🪑 Administración de mobiliario
- 💡 Decoración e iluminación
- 🎵 Equipos multimedia
- 📦 Paquetes de eventos

---

## 👤 Customer Module

Este módulo es utilizado por clientes que desean reservar servicios.

### Funcionalidades:

- 📅 Reservar eventos
- 💳 Realizar pagos
- 📄 Consultar reservaciones
- 📊 Ver historial

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/FrontendPlataformaEventos.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd FrontendPlataformaEventos
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Ejecutar proyecto

```bash
npm run dev
```

---

## 5️⃣ Abrir aplicación

```bash
http://localhost:5173
```
